# 半导体刻蚀技术知识库

本知识库系统介绍半导体刻蚀技术的核心概念、设备架构、工艺原理和故障诊断方法。

## 内容结构

### 第一章：刻蚀基础
- 刻蚀技术的基本概念和分类
- 标准刻蚀工艺步骤
- ALE（原子层刻蚀）技术
- 刻蚀机台整体架构

### 第二章：ICP刻蚀系统
- 电感耦合等离子体（ICP）原理
- 反应腔系统设计
- 线圈与天线系统
- 射频电源系统
- 气体输送系统
- 真空系统
- 温度控制系统
- 废气处理系统

### 第三章：CCP刻蚀系统
- 电容耦合等离子体（CCP）原理
- 电极系统设计
- 射频电源系统
- 等离子体特性
- 气体输送系统
- 刻蚀形貌控制
- 典型应用案例

### 第四章：传感器与故障诊断
- 过程参数传感器
- 终点检测系统
- 故障检测与分类（FDC）
- 预测性维护

## 本地构建

### 安装依赖

```bash
pip install -r requirements.txt
```

### 构建HTML文档

```bash
make html
```

构建完成后，文档位于 `build/html/` 目录。

### 构建PDF文档（需要LaTeX环境）

```bash
make latexpdf
```
