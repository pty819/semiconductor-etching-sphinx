终点检测系统
======

4.3 终点检测系统（Endpoint Detection Systems）
======================================

终点检测是刻蚀工艺控制的核心技术。精确的终点检测确保刻蚀在正确时刻停止，既避免欠刻蚀（Under-etch），又防止过刻蚀（Over-etch）损伤下层结构。

4.3.1 光发射光谱（Optical Emission Spectroscopy, OES）
-----------------------------------------------

OES是半导体刻蚀中最常用的终点检测技术，具有非侵入式、实时、多元素同时监测等优势。

**工作原理**

OES基于以下物理原理：

1. 等离子体中的原子和分子受到电子碰撞激发
2. 激发态的原子/分子退激时发射特征波长的光子
3. 不同元素和分子有特定的发射谱线
4. 通过监测特定波长强度的变化，判断刻蚀进程

*特征发射波长：*

.. list-table:: 常见刻蚀材料的OES监测波长
   :widths: 25 25 50
   :header-rows: 1

   * - 元素/分子
     - 特征波长 (nm)
     - 应用场景
   * - Si
     - 288.2
     - 硅刻蚀终点
   * - F
     - 703.7
     - 含氟气体刻蚀监控
   * - CO
     - 483, 519
     - 氧化物刻蚀（SiO₂）
   * - Cl
     - 837.6
     - 氯基刻蚀监控
   * - SiCl₄
     - 390
     - 多晶硅刻蚀
   * - Br
     - 827.6
     - HBr刻蚀监控
   * - O
     - 777.4
     - 氧等离子体监控
   * - N₂
     - 380-420
     - 氮化物刻蚀

**系统组成**

OES系统由以下关键组件构成：

.. mermaid::

   flowchart LR
       A[等离子体] -->|发射光| B[石英视窗<br/>View Port]
       B --> C[光纤<br/>Fiber Optic]
       C --> D[光谱仪<br/>Spectrometer]
       D --> E[CCD/CMOS<br/>检测器]
       E --> F[数据处理<br/>系统]
       F --> G[终点判断<br/>算法]

*各组件说明：*

1. **石英视窗（View Port）**
   - 材料：高纯度熔融石英
   - 位置：刻蚀腔侧面或顶部
   - 作用：收集等离子体发射光
   - 维护：需要定期清洗防止沉积物影响

2. **光纤（Fiber Optic）**
   - 类型：紫外增强型石英光纤
   - 芯径：200-600 μm
   - 长度：通常2-5米
   - 传输范围：200-1100 nm

3. **光谱仪（Spectrometer）**
   - 分光元件：光栅（1200-2400 lines/mm）
   - 光谱范围：200-1100 nm
   - 光谱分辨率：0.1-2 nm
   - 典型供应商：HORIBA, Ocean Optics, Avantes

4. **检测器（Detector）**
   - 类型：背照式CCD或CMOS
   - 积分时间：10-1000 ms（可调）
   - 动态范围：>16 bit

5. **数据处理系统**
   - 实时光谱分析
   - 峰值识别和强度追踪
   - 终点判断算法

**终点判断算法**

OES终点检测的核心是判断信号变化的算法：

1. **强度阈值法**
   - 当特定波长强度超过/低于设定阈值时触发终点
   - 简单直观，适用于信号变化明显的工艺

2. **斜率变化法**
   - 监测信号强度随时间的变化率（dI/dt）
   - 当斜率发生显著变化时判断终点
   - 更可靠，适用于复杂工艺

3. **比值法**
   - 监测两个波长强度的比值
   - 消除光源波动等共模干扰
   - 提高终点判断的鲁棒性

4. **主成分分析（PCA）法**
   - 对全光谱数据进行降维分析
   - 提取主要变化特征
   - 适用于多元素同时变化的复杂工艺

**优点与局限性**

*优点：*
  - 非侵入式，不干扰等离子体
  - 实时监测，响应速度快
  - 可同时监测多种元素
  - 适用范围广（金属、介质、半导体刻蚀）
  - 系统相对简单，维护成本低

*局限性：*
  - 需要工艺开发确定监测波长
  - 低开放面积（Low Open Area）结构信号弱
  - 视窗污染影响信号质量
  - 对刻蚀深度控制精度有限
  - 不适用于不产生特征发射的工艺

**先进OES技术**

现代OES系统不断演进：

- **全光谱分析**：不再依赖单一波长，而是分析整个光谱的变化模式
- **机器学习辅助终点判断**：使用神经网络、随机森林等算法提高终点判断准确性
- **多腔体同步**：多个刻蚀腔的OES数据联合分析，实现腔体匹配
- **高速OES**：积分时间缩短至毫秒级，适用于脉冲等离子体工艺

4.3.2 激光干涉仪（Laser Interferometry）
---------------------------------

激光干涉仪是另一种重要的终点检测技术，特别适用于透明/半透明薄膜的刻蚀监控。

**工作原理**

激光干涉仪基于光学干涉原理：

1. 单色激光（通常670nm、905nm、980nm）通过视窗照射晶圆表面
2. 激光在薄膜的上表面和下表面分别发生反射
3. 两束反射光产生干涉
4. 随着刻蚀进行，薄膜厚度变化导致光程差变化
5. 干涉信号呈现周期性振荡

*干涉条件：*
  当薄膜厚度满足以下条件时产生相长干涉：
  
  .. math::
     2n·d·cos(θ) = m·λ

  其中：
  - n：薄膜折射率
  - d：薄膜厚度
  - θ：入射角
  - m：干涉级数（整数）
  - λ：激光波长

*刻蚀速率计算：*
  通过计数干涉条纹数量，可以计算刻蚀深度：

  .. math::
     Δd = λ/(2n) × N

  其中N是干涉条纹数量。

**系统参数**

.. list-table:: 激光干涉仪技术参数
   :widths: 35 65
   :header-rows: 1

   * - 参数
     - 典型值
   * - 激光波长
     - 670nm, 905nm, 980nm, 1550nm
   * - 光斑直径
     - 20-60 μm（点测量）/ 1-2 mm（大面积）
   * - 测量精度
     - <1 nm（厚度变化）
   * - 采样频率
     - 1-100 Hz
   * - 适用薄膜
     - 透明/半透明介质膜
   * - 典型供应商
     - HORIBA (LEM系列), Oxford Instruments, Sentronics

**安装位置**

激光干涉仪通过刻蚀腔顶部的专用视窗垂直照射晶圆：

- 视窗直径：≥20mm
- 位置：晶圆中心附近
- 对准：需要精确对准测量区域
- 相机辅助：配备CCD相机辅助光斑定位

**优点与局限性**

*优点：*
  - 精确测量薄膜厚度变化（纳米级分辨率）
  - 直接测量刻蚀深度
  - 适用于透明/半透明薄膜（SiO₂、SiN、光刻胶等）
  - 可检测多层结构的各个界面
  - 不受气体成分变化影响

*局限性：*
  - 需要薄膜在激光波长处透明
  - 不适用于金属或不透明材料
  - 空间分辨率有限（点测量，光斑直径~2mm）
  - 需要已知光学常数（n, k）
  - 多层结构信号复杂，需要建模分析
  - 对表面粗糙度敏感

4.3.3 宽光谱干涉仪（Broadband Interferometry）
--------------------------------------

宽光谱干涉仪（也称为白光干涉仪）是对单波长激光干涉的升级，提供更强大的测量能力。

**与单波长激光干涉的区别**

.. list-table:: 单波长vs宽光谱干涉
   :widths: 30 35 35
   :header-rows: 1

   * - 特性
     - 单波长激光干涉
     - 宽光谱干涉
   * - 光源
     - 单色激光（670/980nm）
     - 宽光谱光源（卤素灯/氙灯）
   * - 光谱范围
     - 单一波长
     - 400-1100 nm（连续光谱）
   * - 测量能力
     - 厚度变化
     - 厚度 + 光学常数
   * - 多层结构
     - 有限（信号复杂）
     - 强（多波长解析）
   * - 材料适用性
     - 需要透明
     - 更广泛
   * - 空间分辨率
     - 点测量
     - 点/面测量

**工作原理**

宽光谱干涉仪使用宽光谱光源（通常是卤素灯或氙灯），同时测量多个波长的干涉信号：

1. 宽光谱光照射到薄膜表面
2. 不同波长在不同膜厚处产生相长/相消干涉
3. 反射光谱包含薄膜厚度信息
4. 通过光谱拟合算法提取薄膜参数

**多波长同时测量的优势**

- **更高的测量精度**：多个波长提供冗余信息，提高拟合精度
- **多层薄膜解析**：不同波长对不同层敏感，可同时解析多层结构
- **光学常数提取**：通过全光谱拟合可同时确定折射率n和消光系数k
- **减少模糊性**：避免单波长干涉中级数判定的模糊性

**适用场景**

- 多层介质膜刻蚀监控
- 高深宽比结构（如3D NAND）
- 复杂膜系结构（如DARC、ARC多层）
- 需要同时测量厚度和光学常数的应用
- 先进封装中的复杂结构

4.3.4 其他终点检测方法
--------------

除OES和干涉仪外，还有多种终点检测技术适用于特定场景。

**质谱分析（Residual Gas Analysis, RGA）**

*工作原理：*
  通过四极杆质谱仪分析真空系统中的气体成分，监测刻蚀过程中产生的副产物。

*测量参数：*
  - 刻蚀副产物的质谱峰强度
  - 气体成分随时间的变化
  - 背景气体污染水平

*应用场景：*
  - 特殊材料刻蚀终点
  - 腔体清洗终点监控
  - 气体泄漏检测
  - 工艺异常诊断

*优缺点：*
  - 优点：高灵敏度，可识别具体化学成分
  - 缺点：设备昂贵，分析复杂，响应速度相对较慢

**声发射传感器（Acoustic Emission）**

*工作原理：*
  监测刻蚀过程中材料去除产生的声波信号。不同材料刻蚀产生不同的声学特征。

*应用场景：*
  - 金属刻蚀终点
  - 硬掩模刻蚀
  - 特殊材料刻蚀监控

*优缺点：*
  - 优点：响应速度快，不受光学窗口污染影响
  - 缺点：环境噪声干扰，需要复杂的信号处理

**椭偏仪（Ellipsometry）**

*工作原理：*
  通过测量偏振光在样品表面反射后偏振状态的变化，确定薄膜厚度和光学常数。

*原位椭偏仪（In-situ Ellipsometry）：*
  集成在刻蚀腔中，实时监测薄膜厚度变化。

*测量参数：*
  - 薄膜厚度（精度<0.1nm）
  - 折射率n和消光系数k
  - 薄膜成分和密度信息

*应用场景：*
  - 高精度刻蚀深度控制
  - 薄膜性质实时监控
  - 工艺开发和研究

*优缺点：*
  - 优点：极高的测量精度，可同时获取多种参数
  - 缺点：系统复杂昂贵，测量面积有限，对表面粗糙度敏感

**等离子体阻抗监控（Plasma Impedance Monitoring, PIM）**

*工作原理：*
  通过VI探针监测等离子体阻抗的变化，当刻蚀穿过界面时阻抗发生特征性变化。

*应用场景：*
  - 刻蚀终点检测（OES的补充手段）
  - 腔体清洗终点
  - 腔体状态监测

*优缺点：*
  - 优点：非侵入式，利用现有VI探针，无需额外硬件
  - 缺点：信号解释复杂，需要经验积累
