典型CCP机台包括：

- **Lam Research Flex系列**：介质刻蚀主力，小体积限制等离子体设计，支持ALE
- **AMAT Centura Etch**：200/300mm平台，陶瓷ESC，Y₂O₃涂层腔体
- **AMAT Producer Etch**：Twin Chamber设计，高产出密度
- **TEL Twinspace**：介质刻蚀平台

各区域工艺特点：

* **A区（中心区）**：
  - 晶圆中央区域
  - 等离子体密度最高
  - 刻蚀速率通常较快
  - 需要补偿以保持均匀性

* **B区（中间区）**：
  - 晶圆主体区域
  - 主要刻蚀发生区
  - 工艺目标区域

* **C区（边缘区）**：
  - 晶圆最外缘
  - 接近聚焦环
  - 需要聚焦环电场修正

* **D区（转角区）**：
  - 晶圆到聚焦环过渡
  - 电场扭曲
  - 容易出现CD偏差
  - 关键工艺控制区域

* **E区（腔壁区）**：
  - 聚焦环外侧到腔壁
  - 等离子体密度快速衰减
  - 腔壁材料保护

多区Showerhead的参数：

* **分区数量**：2-3区（最常见），也有4-5区
* **每区独立MFC控制**
* **每区独立压力控制**
* **每区独立温度控制**

为解决边缘温度损失，ESC采用多区加热：

* **径向分区**：独立控制不同径向位置的加热功率
* **通常4-6区**：从中心到边缘
* **典型温差**：±0.5°C（多区vs单区可达±1.5°C）
* **效果**：改善均匀性、减少CD偏差

**晶圆ESC的径向气体流量**：

He背冷的径向控制同样重要：

* **中心区**：低He流量
* **边缘区**：高He流量（散热需求大）
* **多区控制**：每个区独立MFC

**部件间的相互配合**：

CCP刻蚀机各部件协同工作：

1. **气体路径**：气瓶 → 阀箱 → MFC → 混气室 → 气体分配盘 → Showerhead多孔板 → 等离子体区域
2. **RF路径**：AC电源 → RF电源 → 匹配网络 → 电极 → 等离子体
3. **真空路径**：腔体 → 节流阀 → 挡板 → 分子泵 → 干泵 → 排气处理
4. **冷却路径**：Chiller → 冷却通道 → ESC → 加热器 → He背冷 → 晶圆
5. **控制路径**：上位机 → 工艺控制器 → 各子系统控制器 → 执行机构

这一复杂的协同工作需要精密的控制系统支撑，将在第4章详细讨论。
