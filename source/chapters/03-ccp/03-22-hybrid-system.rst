CCP与ICP混合系统
-----------

现代先进节点工艺中，CCP和ICP常常在同一产线中协同使用。

**协同工艺流程**：

典型的BEOL工艺：

1. **CVD沉积介质层**
2. **PECVD沉积硬掩模**
3. **光刻定义图案**
4. **CCP刻蚀打开硬掩模** ← 高选择比、精确形貌
5. **ICP刻蚀高深宽比通孔** ← 高速率、高各向异性
6. **Strip去胶**
7. **Wet Clean清洗**
8. **金属填充**

这种协同使用充分发挥了两种设备各自的优势。

**Multi-Chamber Cluster Tool**：

在一台Cluster Tool上同时集成CCP和ICP腔体：

* **效率高**：无需破真空传送
* **污染少**：避免大气暴露
* **良率高**：减少氧化和颗粒

例如：

* AMAT Producer系列：Twin Chamber设计，可灵活配置CCP和ICP
* Lam 2300系列：集成Kiyo（ICP）和Flex（CCP）
* TEL Triase+：集成多种刻蚀腔体

**Lam Research的Flex + Kiyo协同**：

* **Flex**：介质刻蚀（CCP），高选择比、低损伤
* **Kiyo**：导体刻蚀（ICP），高密度等离子体
* **Syndion**：TSV刻蚀，专为高深宽比设计

这种协同工艺实现了关键应用的最高精度控制。
