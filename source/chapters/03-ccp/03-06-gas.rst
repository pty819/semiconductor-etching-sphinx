气体输送系统
------

气体输送系统是CCP刻蚀机的关键子系统，负责将多种工艺气体精确、混合均匀地输送到反应腔室，对刻蚀均匀性、选择比和重复性有重要影响。

**气源柜（Gas Cabinet）**：

气源柜是工艺气体的储存和初始处理区域：

* **气体分类储存**：毒性气体（Cl₂）、易燃气体（SiH₄）、惰性气体分别存放
* **自动切换面板（AT-Switch）**：双瓶自动切换，保证供气连续
* **气体纯化器**：去除气体中的水分和杂质
* **尾气收集系统**：收集气瓶残余气体

安全功能：

* **泄漏检测器（GLD）**：H₂、SiH₄等可燃气体监测
* **毒性气体监测器**：HF、HCl、Cl₂监测
* **通风系统**：保证空气清洁

**阀箱（Valve Box）**：

阀箱是气体管路的安全控制中心：

* **手动截止阀（MV）**：维护时隔离气源
* **气动截止阀（AV）**：远程自动控制
* **过流阀（OFV）**：流量超限时自动关闭
* **欠压阀（LPV）**：压力过低时报警
* **单向阀（CV）**：防止气体倒流
* **排空阀（PV）**：维修时排空气体

典型操作顺序：

1. 打开MV
2. 检查管路压力
3. 打开AV
4. 确认MFC流量正常
5. 工艺完成后，按相反顺序关闭

**MFC（Mass Flow Controller）**：

MFC是气体流量的精确控制核心。

工作原理：

1. 气体通过层流元件产生压差
2. 压差由差压传感器测量
3. 微处理器将压差转换为流量
4. 比较测量值与设定值
5. 通过PID调节阀控制流量

关键参数：

* **精度**：±0.5-1%设定值
* **重复性**：±0.2%
* **响应时间**：<1秒（典型），<200ms（快速响应型）
* **流量范围**：sccm到slm
* **耐压**：通常100 psi以上
* **材质**：316L不锈钢、哈氏合金
* **密封**：金属对金属密封（VCR/ConFlat）

常用MFC品牌：

* HORIBA（日本）
* MKS（美国）
* Fujikin（日本）
* Brooks（美国）
* Sevenstar（中国）

**混气室（Gas Manifold）**：

混气室确保多种气体在进入腔体前均匀混合。

设计要求：

* **混合长度**：通常是管径的10-20倍
* **内部结构**：可加入挡板或扰流元件
* **死体积最小化**：避免滞后和残留
* **材质**：与MFC相同的耐腐蚀材料

**Showerhead气体分配**：

Showerhead是气体均匀分配到晶圆的关键部件，详细结构见电极系统章节。

补充说明：

* **多区独立控制**：现代Showerhead可分为2-3个气流区，中心和边缘独立调节流量
* **加热功能**：Showerhead集成加热器，防止聚合物沉积堵塞气体孔
* **清洁功能**：定期使用O₂ plasma清洁，恢复流量

**气体管路工程**：

气体管路设计是CCP刻蚀机的精密工程：

* **管径选择**：1/4"或1/2"不锈钢管（EP级电抛光）
* **连接方式**：VCR（真空管接头）或ConFlat法兰
* **吹扫系统**：N₂吹扫保护管路
* **压力监测**：实时管路压力传感器
* **流量监测**：双重流量计（主计量+校核）

**MFC校准和维护**：

MFC需要定期校准确保精度：

* **定期校准**：通常每6-12个月
* **零点漂移**：环境温度变化影响
* **满量程漂移**：长期使用导致
* **漏气检测**：定期检漏

维护要点：

* **预防性维护**：定期清洁和更换密封件
* **备件管理**：常用MFC备件库存
* **性能趋势**：记录流量精度的历史趋势
* **预防性更换**：使用小时数达到后强制更换

**关键工程问题**：

气体输送系统的常见问题：

* **压力波动**：气源不稳定 → 加稳压阀
* **流量不稳**：MFC校准漂移 → 重新校准
* **响应滞后**：管路过长 → 优化管路设计
* **气体污染**：管路清洗不彻底 → 定期清洗
* **MFC堵塞**：工艺产物残留 → 清洁或更换
* **混合不均**：混气室设计不当 → 改进设计
