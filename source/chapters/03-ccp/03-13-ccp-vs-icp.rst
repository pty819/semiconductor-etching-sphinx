CCP vs ICP应用选择
==================

在选择刻蚀设备时，需要根据具体的工艺需求、材料特性和结构要求来决定使用CCP还是ICP系统。两者各有优势，往往在同一产线中协同使用。

核心选择标准
------------

.. list-table:: CCP与ICP应用场景对比
   :header-rows: 1
   :widths: 30 35 35

   * - 应用场景
     - CCP优势
     - ICP优势
   * - 介质刻蚀（SiO₂、Si₃N₄）
     - ✓ 高选择比，形貌控制好
     - ✗ 需要额外优化
   * - 硅刻蚀（Poly-Si、单晶硅）
     - ✗ 速率较低
     - ✓ 高刻蚀速率，高深宽比
   * - 金属刻蚀（Al、W、Cu）
     - ✗ 部分应用受限
     - ✓ 主流选择
   * - 高深宽比结构（>20:1）
     - ✗ 需要特殊工艺
     - ✓ 天然各向异性
   * - 接触孔/通孔（Contact/Via）
     - ✓ 主流选择
     - ✗ 均匀性挑战
   * - 低损伤要求
     - ✗ 离子能量较高
     - ✓ 低离子能量
   * - ALE（原子层刻蚀）
     - ✓ 精确离子能量控制
     - ✗ 控制复杂度高
   * - STI（浅槽隔离）
     - ✓ 形貌控制精确
     - ✓ 高速率
   * - BEOL（后道工艺）
     - ✓ 低损伤、高选择比
     - ✗ 不适用

**选择原则**：

- 需要高选择比的介质刻蚀 → CCP
- 需要高刻蚀速率或高深宽比 → ICP
- 需要独立控制离子能量和密度 → ICP
- 需要精确形貌控制 → CCP
- 对等离子体损伤敏感 → ICP
