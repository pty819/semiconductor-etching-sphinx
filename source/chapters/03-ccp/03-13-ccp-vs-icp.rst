CCP vs ICP应用选择
==================

CCP vs ICP应用选择
------------------

.. list-table:: CCP与ICP应用场景对比
   :header-rows: 1
   :widths: 30 35 35

   * - 应用场景
     - CCP优势
     - ICP优势
   * - 介质刻蚀（SiO₂、Si₃N₄）
     - ✓ 高选择比，形貌控制好
     - 
   * - 硅刻蚀
     - 
     - ✓ 高刻蚀速率，高深宽比
   * - 金属刻蚀（Al、W）
     - ✓ 部分应用
     - ✓ 主流选择
   * - 高深宽比结构（>20:1）
     - 
     - ✓ 天然各向异性
   * - 接触孔/通孔
     - ✓ 主流选择
     - 
   * - 低损伤要求
     - 
     - ✓ 低离子能量
   * - ALE（原子层刻蚀）
     - ✓ 精确离子能量控制
     - 

**选择原则**：

- 需要高选择比的介质刻蚀 → CCP
- 需要高刻蚀速率或高深宽比 → ICP
- 需要独立控制离子能量和密度 → ICP
- 需要精确形貌控制 → CCP
- 对等离子体损伤敏感 → ICP
